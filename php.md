## PHP 語法基礎
- 用 . 拼接字串
1. 變數前一定要加 $ 
2. 結尾一定要加分號
```php
<?php
$a = 1;
$b ="bbbbb";
echo $a . $b;
?>
```

- if ..else 

1. 少個 ; 也不行

```php
	$score = 70;
	if ($score > 60) {
		echo"pass";
	} else {
		echo"fail";
    }123
```

- for 迴圈
1. 變數前一定要加 $ 
2. 空行 ."<br>" 加HTML標籤
3. 可檢視網頁原始碼
```php
	for ($i = 1; $i <= 10; $i++){
		echo $i."<br>";
    }
```
- Array
1. var_dump($arr) 印出 value, type
2. print_r 印出value
3. array 長度

```php
	$arr = array(1, 2, 3, 4, "prochini");
	echo sizeof($arr) . "<br>";
	var_dump($arr);
	print_r($arr);
```
- function 變數

```php
function add($a, $b){
	return $a +$b;
}
echo add(1, 5);
```
